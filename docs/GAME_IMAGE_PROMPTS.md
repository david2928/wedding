## Tips for Best Results

### Key Phrases:
- **"Simple line art icon"** - Creates clean black outline icon
- **"Centered on cream circular background"** - Icon placement
- **"Abstract watercolor washes behind"** - Keeps background atmospheric but simple
- **"Suggesting [theme]"** - Gives context without detail
- **"Square format"** - Ensures 1:1 ratio

### If Background Too Busy:
